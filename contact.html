Before You Contact Us

To help us assist you better, please consider the following before reaching out:

📋 Check the FAQ

Many common questions and issues are already answered in our Frequently Asked Questions page. You might find your answer there!

🔍 Search Existing Issues

Before opening a new issue, search the existing issues on GitHub to see if someone has already reported the same problem or requested the same feature.

📝 Gather Information

When reporting a bug, please include:

  • Your browser name and version
  • Your operating system
  • Steps to reproduce the issue
  • Expected behavior vs actual behavior
  • Screenshots or error messages (if applicable)

Response Time

We strive to respond to all inquiries as quickly as possible. However, please note that this is a community-maintained open-source project, and response times may vary.

  • Bug Reports: We aim to acknowledge critical bugs within 48 hours
  • Feature Requests: Typically reviewed within 1-2 weeks
  • General Questions: We try to respond within a few days
  • Pull Requests: Code contributions are reviewed as time permits

Community Guidelines

When interacting with the project and community, please follow these guidelines:

  • Be respectful: Treat everyone with courtesy and respect
  • Be constructive: Provide helpful, actionable feedback
  • Be clear: Write clearly and provide context for your questions
  • Be patient: Remember this is a volunteer-maintained project
  • Be inclusive: Welcome contributors of all backgrounds and skill levels

Need Immediate Help?

If you're experiencing issues with the generator, try these quick fixes:

  1. Refresh the page: Sometimes a simple refresh resolves temporary issues
  2. Clear browser cache: Old cached files can cause problems
  3. Try a different browser: Some issues may be browser-specific
  4. Check your connection: Ensure you have internet access (for loading the QR library)
  5. Disable browser extensions: Ad blockers or security extensions may interfere

If none of these help, please check our FAQ or open an issue on GitHub.
